Campaign Landing Pages

The pre-opening campaign landing page was developed with the goal of presenting the consumers with a little bit of information about alot of different aspects of the resort in a central location.

I was able to develop a foundation for these pages so that they could be repurposed with only some simple style and content updates. I was able to use variables in SASS to streamline style updates and the CSS property flexbox to create flexable and dynamic layouts.
